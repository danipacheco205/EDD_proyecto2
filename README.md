# EDD_proyecto2
Se desarrolló un programa que administra hasta 100 artículos científicos guardándolos en la memoria, este programa permite la búsqueda y el análisis de cada una de las investigaciones almacenadas, además de mostrar el cuerpo del resumen, sus autores y la frecuencia de cada palabra clave de esa investigación.
